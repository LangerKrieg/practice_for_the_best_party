В диалоговом окне:
При нажатии на "Загрузить граф..." будет предложено выбрать сохраненный файл.
При нажатии на "OK" создадутся вершины на поле главного окна, диалоговое окно будет закрыто.
При нажатии на "Cancel" будут закрыты все окна программы.
В главном окне:
При нажатии на "Добавить путь" пользователю будет предложено создать ребро между двумя вершинами. При повторном нажатии возможность убирается.
При нажатии на "Удалить путь" пользователю будет предложено удалить ребро между двумя вершинами. При повторном нажатии возможность убирается.
При нажатии на "Удалить мосты" в графе будут удалены все мосты.
При нажатии на "Найти мосты" в графе будут подсвечены все мосты. Функция отключается при повторном нажатии.
При нажатии на "Сохранить..." будет предложено создать save file графа.
